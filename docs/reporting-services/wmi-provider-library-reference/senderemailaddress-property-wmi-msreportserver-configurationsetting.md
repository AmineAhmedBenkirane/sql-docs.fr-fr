# Propri&#233;t&#233; SenderEmailAddress (WMI MSReportServer_ConfigurationSetting)
  Obtient l'adresse utilisée pour l'envoi de messages électroniques à partir du serveur de rapports. En lecture seule.  
  
## Syntaxe  
  
```vb  
Public Dim SenderEmailAddress As String  
```  
  
```csharp  
public string SenderEmailAddress;  
```  
  
## Valeurs de la propriété  
 Objet **String** en lecture seule qui représente l’adresse e-mail utilisée par le serveur de rapports.
