# <a name="attribute-properties---remove-an-attribute-from-a-dimension"></a>Propriétés d’attribut - supprimer un attribut d’une Dimension
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]Pour supprimer un attribut d’une dimension dans [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)], avec le bouton droit de l’attribut dans le **attributs** volet de la **Structure de Dimension** onglet de Dimension Concepteur dans [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)], puis cliquez sur **supprimer**. Dans la boîte de dialogue **Supprimer les objets** , cliquez sur **OK**. L'attribut est supprimé de la dimension, mais ceci n'affecte pas la vue de source de données pour la dimension.
