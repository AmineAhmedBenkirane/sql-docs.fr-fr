## <a name="using-the-jdbc-driver-with-maven-central"></a>Utilisation du pilote JDBC avec Maven Central
Le pilote JDBC peuvent être ajouté à un projet Maven en l’ajoutant en tant que dépendance dans le fichier POM.xml avec le code ci-dessous :

```xml
<dependency>
    <groupId>com.microsoft.sqlserver</groupId>
    <artifactId>mssql-jdbc</artifactId>
    <version>6.2.1.jre8</version>
</dependency>
```
