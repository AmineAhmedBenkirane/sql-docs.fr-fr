# <a name="getboolean-method-sqlservercallablestatement"></a>Méthode getBoolean (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre désigné en tant qu’un **booléenne** valeur.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[getBoolean (int)](../../../connect/jdbc/reference/getboolean-method-int.md)|Récupère la valeur du paramètre désigné en tant qu’un **booléenne** valeur en fonction de l’index de paramètre.|  
|[getBoolean (java.lang.String)](../../../connect/jdbc/reference/getboolean-method-java-lang-string.md)|Récupère la valeur du paramètre désigné en tant qu’un **booléenne** étant donné le nom du paramètre de valeur.|
