# <a name="odbc-driver-for-oracle-conformance-levels"></a>Pilote ODBC pour Oracle les niveaux de conformité
> [!IMPORTANT]  
>  Cette fonctionnalité sera supprimée dans une future version de Windows. Évitez d'utiliser cette fonctionnalité dans de nouveaux travaux de développement, et prévoyez de modifier les applications qui utilisent actuellement cette fonctionnalité. Au lieu de cela, utilisez le pilote ODBC fourni par Oracle.  
  
 ODBC définit deux types de normes de conformité pour les pilotes : la conformité des API standard et la conformité de la grammaire SQL standard. Mise en conformité de l’API fait référence aux fonctions qui prend en charge par un pilote. Mise en conformité SQL fait référence à la grammaire SQL qui prend en charge par le pilote. Chaque mise en conformité standard est constitué de niveaux.  
  
 Cette section fournit des informations de conformité sur le pilote ODBC pour Oracle.
