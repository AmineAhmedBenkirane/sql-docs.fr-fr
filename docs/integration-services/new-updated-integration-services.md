**Configurer le Gestionnaire de connexions ADO.NET**

 **Connexions de données** sélectionner une connexion de données ADO.NET existante dans la liste.

 **Propriétés de connexion de données** afficher les propriétés et valeurs de la connexion de données ADO.NET sélectionnée.

 **Nouvelle** créer une connexion de données ADO.NET à l’aide de la **Gestionnaire de connexions** boîte de dialogue.

 **Supprimer** sélectionner une connexion, puis supprimez-la à l’aide de la **supprimer** bouton.

**Éditeur du Gestionnaire de connexions FTP**

  La boîte de dialogue **Éditeur du gestionnaire de connexions FTP** vous permet de spécifier les propriétés de connexion à un serveur FTP.

> [!IMPORTANT]
>  Le gestionnaire de connexions FTP prend en charge uniquement l'authentification anonyme et l'authentification de base. Il ne prend pas en charge l'authentification Windows.

 Pour en savoir plus sur le Gestionnaire de connexions FTP, consultez [FTP connexion Manager--... /.. / integration-services/connection-manager/ftp-connection-manager.md).

**Options**

 **Nom du serveur** fournir le nom du serveur FTP.

 **Port du serveur** spécifier le numéro de port sur le serveur FTP à utiliser pour la connexion. La valeur par défaut de cette propriété est **21**.

 **Nom d’utilisateur** fournissent un nom d’utilisateur pour accéder au serveur FTP. La valeur par défaut de cette propriété est **anonyme**.

 **Mot de passe** fournir le mot de passe pour accéder au serveur FTP.

 **Délai d’attente (en secondes)** spécifier le nombre de secondes avant l’expiration que prend la tâche. Une valeur égale à **0** indique une durée illimitée. La valeur par défaut de cette propriété est **60**.

 **Utiliser le mode passif** spécifier si le serveur ou le client initie la connexion. Le serveur initie la connexion en mode actif, alors que le client l'initie en mode passif. La valeur par défaut de cette propriété est **mode actif**.

 **Effectue une nouvelle tentative** spécifier le nombre de fois où la tâche tente d’établir une connexion. Une valeur égale à **0** indique un nombre illimité de tentatives.

 **Taille (en Ko) du segment** fournissent une taille de segment en kilo-octets pour la transmission des données.

 **Tester la connexion** après avoir configuré le Gestionnaire de connexions FTP, confirmez que la connexion est viable en cliquant sur **tester la connexion**.

 **Colonnes d’entrée disponibles** en utilisant les cases à cocher, spécifiez les colonnes à convertir en lignes.

 **Nom de** afficher le nom de la colonne d’entrée disponible.

 **Transfert direct** indiquer s’il faut inclure la colonne dans la sortie de transformation.

 **Colonne d’entrée** sélectionnez dans la liste des colonnes d’entrée disponibles pour chaque ligne. Vos sélections se reflètent dans les sélections des cases à cocher de la table **Colonnes d'entrée disponibles** .

 Dans le scénario Unpivot décrit dans [Unpivot Transformation--... /.. /.. / integration-services/data-flow/transformations/unpivot-transformation.md), les colonnes d’entrée sont les **Ham**, **Soda**, **lait**, **Beer**, et **Chips** colonnes.

 **Colonne de destination** fournir un nom pour la colonne de données.

 Dans le scénario Unpivot décrit dans [Unpivot Transformation--... /.. /.. / integration-services/data-flow/transformations/unpivot-transformation.md), la colonne de Destination est la quantité (**Qty**) colonne.

 **Valeur de clé de tableau croisé dynamique** fournir un nom pour la valeur de tableau croisé dynamique. Par défaut, il s'agit du nom de la colonne d'entrée ; vous pouvez néanmoins choisir un nom unique et descriptif.

 Il est possible de spécifier la valeur de cette propriété en utilisant l'expression d'une propriété.

 Dans le scénario Unpivot décrit dans [Unpivot Transformation--... /.. /.. / integration-services/data-flow/transformations/unpivot-transformation.md), les valeurs de tableau croisé dynamique s’affiche dans la nouvelle colonne produit désignée par le **nom de colonne de valeur de clé de tableau croisé dynamique** option, car les valeurs de texte **Ham **, **Soda**, **lait**, **Beer**, et **Chips**.
