# Cat&#233;gorie d&#39;&#233;v&#233;nement &#201;v&#233;nements de session
  La catégorie d'événement Événements de session contient les classes d'événements décrites dans le tableau ci-dessous.  
  
|Classe d'événements|ID d'événement|Description|  
|-----------------|--------------|-----------------|  
|Connexion existante|41|Collecte tous les événements de connexion depuis le début de la trace.|  
|Existing Session|42|Collecte tous les événements de session depuis le début de la trace.|  
|Session Initialize|43|Recueille les événements d'initialisation de session.|  
  
 Pour plus d’informations sur les colonnes associées à chaque classe d’événements Session Events, consultez [Colonnes de données des événements de session](../../analysis-services/trace-events/session-events-data-columns.md).
