# Propri&#233;t&#233; DatabaseServerName (WMI MSReportServer_ConfigurationSetting)
  Spécifie le nom du serveur sur lequel la base de données du serveur de rapports est installée. En lecture/écriture.  
  
## Syntaxe  
  
```vb  
Public Dim DatabaseServerName As String  
```  
  
```csharp  
public string DatabaseServerName;  
```  
  
## Valeurs de la propriété  
 Objet **String** qui représente le nom du serveur.
