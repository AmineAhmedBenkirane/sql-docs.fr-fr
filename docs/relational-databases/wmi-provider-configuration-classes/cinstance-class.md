# <a name="cinstance-class"></a>Classe CInstance
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Le [classe CInstance](../../relational-databases/wmi-provider-configuration-classes/cinstance-class.md) représente une instance d’un [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] client.  
  
 Cette classe peut être utilisée pour gérer les protocoles et les bibliothèques réseau associés à l'instance d'un client [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .
