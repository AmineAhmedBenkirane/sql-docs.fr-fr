# Propri&#233;t&#233;s de NS$&lt;nom du service&gt; (onglet Avanc&#233;)
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] est implémenté comme un service nommé au format **NS$***<nom configuré_utilisateur>*. Si des propriétés personnalisées sont définies, elles apparaissent dans cet onglet, avec les valeurs correspondantes. Il est possible qu'aucune propriété ne soit répertoriée dans cet onglet.
