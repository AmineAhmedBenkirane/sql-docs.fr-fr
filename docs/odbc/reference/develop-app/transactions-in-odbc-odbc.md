# <a name="transactions-in-odbc-odbc"></a>Transactions dans ODBC ODBC
Les transactions dans ODBC sont exécutées au niveau de la connexion ; Autrement dit, lorsqu’une application termine une transaction, il valide ou restaure tout le travail effectué à partir de tous les descripteurs d’instruction sur cette connexion.
