# <a name="-positive-dmx"></a>+ (positif) (DMX)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx_md](../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  Exécute une opération unaire qui retourne la valeur positive d'une expression numérique.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
+ Numeric_Expression  
```  
  
#### <a name="parameters"></a>Paramètres  
 *Numeric_expression*  
 Expression DMX (Data Mining Extensions) valide qui retourne une valeur numérique.  
  
## <a name="return-value"></a>Valeur retournée  
 Valeur dotée du type de données du paramètre spécifié.
