# Afficher des serveurs inscrits dans SQL Server Management Studio
  Cette rubrique explique comment afficher les serveurs inscrits dans [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)].  
  
##  <a name="SSMSProcedure"></a>  
  
#### Pour afficher les serveurs inscrits dans SQL Server Management Studio  
  
1.  Dans [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)], dans le menu **Affichage** , cliquez sur **Serveurs inscrits**.  
  
     Par défaut, l'écran Serveurs inscrits apparaît au-dessus de l'Explorateur d'objets dans la fenêtre principale.
