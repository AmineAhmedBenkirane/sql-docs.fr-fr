# <a name="getbyte-method-sqlservercallablestatement"></a>Méthode getByte (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre désigné en tant qu’un **octets** valeur.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[getByte (int)](../../../connect/jdbc/reference/getbyte-method-int.md)|Récupère la valeur du paramètre désigné en tant qu’un **octets** valeur en fonction de l’index de paramètre.|  
|[getByte (java.lang.String)](../../../connect/jdbc/reference/getbyte-method-java-lang-string.md)|Récupère la valeur du paramètre désigné en tant qu’un **octets** étant donné le nom du paramètre de valeur.|
