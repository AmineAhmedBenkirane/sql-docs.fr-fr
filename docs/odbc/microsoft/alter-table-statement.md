# <a name="alter-table-statement"></a>Instruction ALTER TABLE
Lorsque le Microsoft Access, dBASE ou Paradoxdriver est utilisé, l’instruction ALTER TABLE est pris en charge avec les clauses de définition de contrainte table add et drop. (Les instructions ALTER TABLE ne sont pas pris en charge pour les pilotes Microsoft Excel ou texte.)  
  
> [!NOTE]  
>  Lorsque vous utilisez le pilote Paradox sans implémenter le moteur de base de données Borland, les instructions ALTER TABLE ne sont pas pris en charge ; uniquement lire et ajoutez les instructions sont autorisées.
