# Propri&#233;t&#233; EditionID (WMI MSReportServer_Instance)
  Retourne l'édition du serveur de rapports. En lecture seule.  
  
## Syntaxe  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## Valeur de propriété  
 Valeur **uint** qui contient l’édition du serveur de rapports.
