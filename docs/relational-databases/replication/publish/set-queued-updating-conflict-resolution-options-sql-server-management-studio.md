# d&#233;finir des options de r&#233;solution des conflits de mise &#224; jour en attente (SQL Server Management Studio)
  Définir des conflits pour les publications en file d’attente de prise en charge des abonnements mis à jour sur les options de résolution de la **Options d’abonnement** page de le **Propriétés de la Publication - \< Publication>** boîte de dialogue. Pour plus d'informations sur l'accès à cette boîte de dialogue, consultez [View and Modify Publication Properties](../../../relational-databases/replication/publish/view-and-modify-publication-properties.md).  
  
### Pour définir les options de résolution des conflits de mise à jour en attente  
  
1.  Sur le **Options d’abonnement** page de le **Propriétés de la Publication - \< Publication>** boîte de dialogue, sélectionnez une de ces valeurs pour le **la stratégie de résolution de conflit** option :  
  
    -   **Conserver la modification apportée au serveur de publication**  
  
    -   **Conserver la modification apportée à l'abonné**  
  
    -   **Réinitialiser l'abonnement**  
  
2.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]
