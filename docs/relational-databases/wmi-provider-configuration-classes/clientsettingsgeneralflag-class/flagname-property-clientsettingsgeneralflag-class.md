# <a name="flagname-property-clientsettingsgeneralflag-class"></a>Propriété FlagName (classe ClientSettingsGeneralFlag)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Obtient le nom de l’indicateur.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object.FlagName [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *objet*  
 Objet de [classe ClientSettingsGeneralFlag](../../../relational-databases/wmi-provider-configuration-classes/clientsettingsgeneralflag-class/clientsettingsgeneralflag-class.md) qui représente un indicateur général pour les paramètres du serveur.  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Valeur de chaîne qui spécifie le nom de l'indicateur.
