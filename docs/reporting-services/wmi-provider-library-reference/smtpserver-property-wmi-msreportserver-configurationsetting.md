# Propri&#233;t&#233; SMTPServer (WMI MSReportServer_ConfigurationSetting)
  Obtient la propriété du serveur SMTP à partir du fichier de configuration du serveur de rapports. En lecture seule.  
  
## Syntaxe  
  
```vb  
Public Dim SMTPServer As String  
```  
  
```csharp  
public string SMTPServer;  
```  
  
## Valeurs de la propriété  
 Objet **String** en lecture seule contenant la valeur de la propriété **SMTPServer** du fichier RSReportServer.config.
