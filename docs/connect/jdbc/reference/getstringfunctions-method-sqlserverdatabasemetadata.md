# <a name="getstringfunctions-method-sqlserverdatabasemetadata"></a>Méthode getStringFunctions (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère une liste séparée par des virgules de **chaîne** fonctions qui sont disponibles avec cette base de données.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public java.lang.String getStringFunctions()  
```  
  
## <a name="return-value"></a>Valeur retournée  
 A **chaîne** qui contient le **chaîne** fonctions.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode getStringFunctions est spécifiée par la méthode getStringFunctions dans l’interface java.sql.DatabaseMetaData.
