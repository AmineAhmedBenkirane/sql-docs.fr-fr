# Propri&#233;t&#233; IsWebServiceEnabled (WMI MSReportServer_ConfigurationSetting)
  Indique si le service Web Report Server est activé. En lecture seule.  
  
## Syntaxe  
  
```vb  
Public Dim IsWebServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWebServiceEnabled;  
```  
  
## Valeurs de la propriété  
 Valeur **booléenne** en lecture seule. La valeur **true** indique que le service web Report Server est activé.
