# <a name="sql-server-clr-object"></a>SQL Server, objet CLR
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] L’objet **SQLServer:CLR** fournit des compteurs qui permettent de surveiller l’exécution du CLR (Common Language Runtime) dans [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
 Ce tableau décrit les compteurs [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] **de** .  
  
|Compteurs CLR|Description|  
|------------------|-----------------|  
|**Exécution CLR**|Durée d'exécution totale dans CLR (microsecondes)|
