# <a name="clearwarnings-method-sqlserverresultset"></a>Méthode clearWarnings (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Efface tous les avertissements signalés sur cet [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objet.  
  
> [!NOTE]  
>  Cette méthode n’est pas implémentée actuellement par le [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]. Si elle est appelée, elle renvoie toujours la valeur Null.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public void clearWarnings()  
```  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode clearWarnings est spécifiée par la méthode clearWarnings dans l’interface java.sql.ResultSet.
