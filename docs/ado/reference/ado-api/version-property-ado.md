# <a name="version-property-ado"></a>Propriété version (ADO)
Indique le numéro de version d’ADO.  
  
## <a name="return-value"></a>Valeur retournée  
 Retourne un **chaîne** valeur qui indique la version.  
  
## <a name="remarks"></a>Notes  
 Utilisez le **Version** propriété pour retourner le numéro de version de l’implémentation ADO.  
  
 La version du fournisseur sera disponible en tant que propriété dynamique dans le [propriétés](../../../ado/reference/ado-api/properties-collection-ado.md) collection.
