# <a name="getmultisubnetfailover-method-sqlserverdatasource"></a>Méthode getMultiSubnetFailover (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Retourne la valeur de la **multiSubnetFailover** propriété de connexion.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
public boolean getMultiSubnetFailover();  
```  
  
## <a name="return-value"></a>Valeur retournée  
 Retourne True ou False, en fonction du paramètre actuel de la propriété de connexion.  
  
## <a name="remarks"></a>Notes  
 Pour plus d’informations sur la **multiSubnetFailover** propriété de connexion, consultez [définissant les propriétés de connexion](../../../connect/jdbc/setting-the-connection-properties.md).
