# <a name="getsqlxml-method-sqlservercallablestatement"></a>Méthode getSQLXML (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre désigné en tant qu'objet java.sql.SQLXML.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[Méthode getSQLXML &#40;int&#41;](../../../connect/jdbc/reference/getsqlxml-method-int.md)|Récupère la valeur du paramètre désigné en tant qu’objet SQLXML fonction de l’index de paramètre.|  
|[Méthode getSQLXML &#40;java.lang.String&#41;](../../../connect/jdbc/reference/getsqlxml-method-java-lang-string.md)|Récupère la valeur du paramètre désigné en tant qu’objet SQLXML étant donné le nom de paramètre.|
