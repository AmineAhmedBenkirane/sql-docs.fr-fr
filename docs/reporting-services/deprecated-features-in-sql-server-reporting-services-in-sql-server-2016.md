# Fonctions d&#233;conseill&#233;es de SQL Server Reporting Services dans SQL Server 2016
  Cette rubrique décrit les fonctionnalités déconseillées de [!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)] . Les fonctionnalités sont toujours disponibles dans la version dans laquelle elles sont déconseillées ; toutefois, leur suppression est planifiée dans une version future de [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]. N’utilisez pas de fonctions déconseillées dans les nouvelles applications.  
  
## Fonctionnalités non prises en charge dans la prochaine version de SQL Server Reporting Services  
 Les fonctionnalités suivantes de [!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)] ne seront pas prises en charge dans la prochaine version de [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]. Évitez d’utiliser ces fonctionnalités dans vos nouveaux développements et modifiez dès que possible les applications qui y ont recours.  
  
|Catégorie|Fonctionnalité déconseillée|
|--------------|------------------------| 
|Serveur de rapports|Convertisseur HTML4.0. Utilisez le convertisseur HTML5.|
