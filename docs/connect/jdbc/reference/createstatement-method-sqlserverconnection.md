# <a name="createstatement-method-sqlserverconnection"></a>createStatement (méthode) (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Crée un [SQLServerStatement](../../../connect/jdbc/reference/sqlserverstatement-class.md) objet pour l’envoi d’instructions SQL à la base de données.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[createStatement ()](../../../connect/jdbc/reference/createstatement-method.md)|Crée un objet SQLServerStatement pour l'envoi d'instructions SQL à la base de données.|  
|[createStatement (int, int)](../../../connect/jdbc/reference/createstatement-method-int-int.md)|Crée un objet SQLServerStatement qui génère des objets SQLServerResultSet avec la concurrence et le type donnés.|  
|[createStatement (int, int, int)](../../../connect/jdbc/reference/createstatement-method-int-int-int.md)|Crée un objet SQLServerStatement qui génère des objets SQLServerResultSet avec le type, la concurrence et la fonctionnalité de mise en attente donnés.|
