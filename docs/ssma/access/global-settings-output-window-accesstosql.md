# <a name="global-settings-output-window-accesstosql"></a>Paramètres globaux (fenêtre de sortie) (AccessToSQL)
Utilisez le **paramètres globaux** boîte de dialogue pour spécifier les paramètres de SSMA.  
  
Pour accéder aux paramètres de la fenêtre Sortie, sur le **outils** menu, sélectionnez **paramètres globaux**, cliquez sur **GUI** en bas du volet gauche, puis **fenêtre sortie**.  
  
## <a name="options"></a>Options  
**Afficher le volet de la liste d’erreurs**  
Sélectionnez **True** si vous souhaitez SSMA pour afficher le volet de la liste d’erreurs lorsque des messages.  
  
**Afficher le volet de sortie**  
Sélectionnez **True** si vous souhaitez SSMA pour afficher le volet de sortie lorsque des messages.
