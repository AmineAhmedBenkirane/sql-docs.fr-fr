# <a name="show-advanced-options-server-configuration-option"></a>show advanced options (option de configuration de serveur)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]

  Utilisez l’option **show advanced options** pour afficher les options avancées de la procédure stockée système **sp_configure** . Si vous affectez la valeur 1 à l’option **show advanced options** , vous pouvez dresser la liste des options avancées à l’aide de **sp_configure**. La valeur par défaut est 0.  
  
 Le paramètre prend effet immédiatement (sans redémarrage du serveur).
