# <a name="rename-a-database-diagram-visual-database-tools"></a>Renommer un schéma de base de données (Visual Database Tools)
Vous pouvez renommer un schéma de base de données dans l'Explorateur d'objets. Nécessite l'autorisation ALTER sur la base de données.  
  
### <a name="to-rename-a-database-diagram"></a>Pour renommer un schéma de base de données  
  
1.  Dans l’Explorateur d’objets, développez le dossier **Schémas de base de données** .  
  
2.  Cliquez avec le bouton droit sur le schéma de base de données que vous souhaitez renommer.  
  
3.  Dans le menu contextuel, sélectionnez **Renommer** .  
  
    Une zone d'édition s'ouvre autour du nom du schéma dans l'Explorateur d'objets.  
  
4.  Entrez le nouveau nom du schéma de base de données et appuyez sur **Entrée**.  
  
    Le schéma de base de données apparaît sous son nouveau nom dans l'Explorateur d'objets.
