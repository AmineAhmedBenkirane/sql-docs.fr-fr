# <a name="export-as-policy-dialog-box"></a>Boîte de dialogue Exporter en tant que stratégie
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Utilisez cette boîte de dialogue pour enregistrer un jeu de propriétés de facette de la Gestion basée sur des stratégies en tant que stratégie. Les stratégies peuvent être enregistrées dans un fichier XML ou directement dans l'instance du [!INCLUDE[ssDE](../../includes/ssde-md.md)]actuelle.  
  
## <a name="options"></a>Options  
 **Nom de la définition de stratégie**  
 Tapez un nom pour la stratégie.  
  
 **Nom de la condition**  
 Tapez un nom pour la condition.  
  
 **Vers le serveur local**  
 Sélectionnez cette option pour enregistrer la stratégie sur l'instance de serveur locale.  
  
 **Vers le fichier**  
 Pour enregistrer la stratégie en tant que fichier XML, tapez le nom et le chemin d’accès du fichier ou utilisez le bouton d’exploration (**...**).
