# <a name="length-method-sqlserverclob"></a>Méthode length (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Retourne le nombre de caractères dans l'objet CLOB.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public long length()  
```  
  
## <a name="return-value"></a>Valeur retournée  
 Longueur de l'objet CLOB en nombre de caractères.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode de longueur est spécifiée par la méthode de longueur dans l’interface java.sql.Clob.
