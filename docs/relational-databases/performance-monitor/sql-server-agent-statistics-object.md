# <a name="sql-server-agent-statistics-object"></a>Agent SQL Server, Objet Statistics
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] L’objet de performance **Statistics** de SQL Server Agent contient des compteurs de performances qui fournissent des informations au sujet de cet agent.  
  
 Le tableau ci-dessous décrit le compteur de **SQLAgent:Statistics** .  
  
|Nom|Description|  
|----------|-----------------|  
|**Redémarrages de SQL Server**|Nombre de redémarrages réussis de Microsoft SQL Server effectués par l'Agent SQL Server depuis le dernier démarrage de ce dernier.|
