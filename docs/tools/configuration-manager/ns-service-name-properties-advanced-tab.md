# <a name="nsltservice-namegt-properties-advanced-tab"></a>NS$&lt;nom du service&gt; propriétés (onglet Avancé)
  [!INCLUDE[ssNS](../../includes/ssns-md.md)]est implémenté en tant que service nommé au format **NS$***< nom configuré_utilisateur >*. Si des propriétés personnalisées sont définies, elles apparaissent dans cet onglet, avec les valeurs correspondantes. Il est possible qu'aucune propriété ne soit répertoriée dans cet onglet.
