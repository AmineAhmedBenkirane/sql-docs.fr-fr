# <a name="getcharacterstream-method-sqlserverclob"></a>Méthode getCharacterStream (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère le **Clob** valeur désignée par l’objet java.sql.Clob en tant qu’objet java.io.Reader.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[Méthode getCharacterStream&#40;&#41;](../../../connect/jdbc/reference/getcharacterstream-method.md)|Retourne les données Clob en tant qu'objet java.io.Reader ou que flux de caractères.|  
|[Méthode getCharacterStream &#40;long, long&#41;](../../../connect/jdbc/reference/getcharacterstream-method-long-long.md)|Retourne les données Clob en tant qu’objet java.io.Reader ou en tant que flux de caractères avec la position spécifiée et la longueur.|
