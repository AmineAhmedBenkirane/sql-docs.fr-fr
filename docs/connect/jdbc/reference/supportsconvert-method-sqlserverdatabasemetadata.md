# <a name="supportsconvert-method-sqlserverdatabasemetadata"></a>Méthode supportsConvert (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère les informations déterminant si cette base de données prend en charge la fonction CONVERT entre les types SQL.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[supportsConvert ()](../../../connect/jdbc/reference/supportsconvert-method.md)|Récupère les informations déterminant si cette base de données prend en charge la fonction CONVERT entre les types SQL.|  
|[supportsConvert (int, int)](../../../connect/jdbc/reference/supportsconvert-method-int-int.md)|Récupère les informations déterminant si cette base de données prend en charge la fonction CONVERT pour deux types SQL donnés.|
