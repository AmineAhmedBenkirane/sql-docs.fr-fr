# <a name="deprecation-event-category"></a>catégorie d'événement Deprecation
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)]La catégorie d'événement **Deprecation** contient des événements de dépréciation généraux.  
  
## <a name="in-this-section"></a>Dans cette section  
  
|Rubrique|Description|  
|-----------|-----------------|  
|[Classe d'événements Deprecation Announcement](../../relational-databases/event-classes/deprecation-announcement-event-class.md)|Indique qu'une fonctionnalité sera retirée dans une future version de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].|  
|[Classe d'événements Deprecation Final Support](../../relational-databases/event-classes/deprecation-final-support-event-class.md)|Indique qu'une fonctionnalité sera retirée à compter de la prochaine version majeure de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].|
