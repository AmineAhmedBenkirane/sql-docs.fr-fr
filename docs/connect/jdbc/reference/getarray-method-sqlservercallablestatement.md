# <a name="getarray-method-sqlservercallablestatement"></a>Méthode getArray (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre désigné en tant qu’objet de tableau.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[getArray (int)](../../../connect/jdbc/reference/getarray-method-int.md)|Récupère la valeur du paramètre désigné en tant qu’objet de tableau, l’index de paramètre.|  
|[getArray (java.lang.String)](../../../connect/jdbc/reference/getarray-method-java-lang-string.md)|Récupère la valeur du paramètre désigné en tant qu’objet de tableau, étant donné le nom de paramètre.|
