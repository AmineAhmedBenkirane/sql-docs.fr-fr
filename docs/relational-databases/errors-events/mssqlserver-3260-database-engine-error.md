## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|SQL Server|  
|ID d'événement|3260|  
|Source de l'événement|MSSQLSERVER|  
|Composant|SQLEngine|  
|Nom symbolique|DMPIO_BUFFER_OVERFLOW|  
|Texte du message|Un tampon interne est saturé.|
