# <a name="ado-md-object-model"></a>Modèle objet ADO MD
Cette rubrique décrit comment les objets sont représentés et liées dans ADO MD.  
  
 ![ADO MD Object Model](../../../ado/reference/ado-md-api/media/ado_md_object_model.gif "ADO_MD_object_model")  
  
 Le [axe](../../../ado/reference/ado-md-api/axis-object-ado-md.md) et [cellule](../../../ado/reference/ado-md-api/cell-object-ado-md.md) chacun des objets ont une [Positions](../../../ado/reference/ado-md-api/positions-collection-ado-md.md) collection.  
  
 Le [niveau](../../../ado/reference/ado-md-api/level-object-ado-md.md) et [Position](../../../ado/reference/ado-md-api/position-object-ado-md.md) chacun des objets ont une [membres](../../../ado/reference/ado-md-api/members-collection-ado-md.md) collection.  
  
 Le [axe](../../../ado/reference/ado-md-api/axis-object-ado-md.md), [cellule](../../../ado/reference/ado-md-api/cell-object-ado-md.md), [ensemble de cellules](../../../ado/reference/ado-md-api/cellset-object-ado-md.md), [CubeDef](../../../ado/reference/ado-md-api/cubedef-object-ado-md.md), [Dimension](../../../ado/reference/ado-md-api/dimension-object-ado-md.md), [hiérarchie](../../../ado/reference/ado-md-api/hierarchy-object-ado-md.md), [Niveau](../../../ado/reference/ado-md-api/level-object-ado-md.md), et [membre](../../../ado/reference/ado-md-api/member-object-ado-md.md) chacun des objets ont une ADO standard [propriétés](../../../ado/reference/ado-api/properties-collection-ado.md) collection.
