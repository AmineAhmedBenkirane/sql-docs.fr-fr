# Propri&#233;t&#233; WindowsServiceIdentityActual (WMI MSReportServer_ConfigurationSetting)
  Identité sous laquelle le service Windows Report Server s'exécute. En lecture seule.  
  
## Syntaxe  
  
```vb  
Public Dim WindowsServiceIdentityActual As String  
```  
  
```csharp  
public string WindowsServiceIdentityActual;  
```  
  
## Valeurs de la propriété  
 Objet **String** contenant l’identité sous laquelle le service Windows Report Server s’exécute.
