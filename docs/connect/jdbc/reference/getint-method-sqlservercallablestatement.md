# <a name="getint-method-sqlservercallablestatement"></a>Méthode getInt (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre désigné en tant qu’un **int** dans le langage de programmation Java.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[getInt (int)](../../../connect/jdbc/reference/getint-method-int.md)|Récupère la valeur du paramètre désigné en tant qu’un **int** en fonction de l’index de paramètre de langage de programmation Java.|  
|[getInt (java.lang.String)](../../../connect/jdbc/reference/getint-method-java-lang-string.md)|Récupère la valeur du paramètre désigné en tant qu’un **int** en fonction du nom du paramètre de langage de programmation Java.|
