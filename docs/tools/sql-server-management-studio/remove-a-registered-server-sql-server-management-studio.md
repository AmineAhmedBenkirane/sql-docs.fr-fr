# Supprimer un serveur inscrit (SQL Server Management Studio)
  Cette rubrique explique comment supprimer un serveur des serveurs inscrits dans [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] à l'aide de [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)].  
  
##  <a name="SSMSProcedure"></a>  
  
#### Pour supprimer un serveur inscrit  
  
1.  Dans Serveurs inscrits, cliquez avec le bouton droit sur un serveur, puis cliquez sur **Supprimer**.  
  
2.  Dans la boîte de dialogue **Demande de confirmation de suppression** , cliquez sur **Oui**.
