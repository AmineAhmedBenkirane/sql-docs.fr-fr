# <a name="getstring-method-sqlservercallablestatement"></a>Méthode getString (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre désigné en tant qu’un **chaîne** dans le langage de programmation Java.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[getString (int)](../../../connect/jdbc/reference/getstring-method-int.md)|Récupère la valeur du paramètre désigné en tant qu’un **chaîne** en fonction de l’index de paramètre de langage de programmation Java.|  
|[getString (java.lang.String)](../../../connect/jdbc/reference/getstring-method-java-lang-string.md)|Récupère la valeur du paramètre désigné en tant qu’un **chaîne** en fonction du nom du paramètre de langage de programmation Java.|
