# <a name="source-property-ado-md"></a>Source, propriété (ADO MD)
Indique la source de données dans le [ensemble de cellules](../../../ado/reference/ado-md-api/cellset-object-ado-md.md).  
  
## <a name="settings-and-return-values"></a>Paramètres et valeurs de retour  
 Définit ou retourne un **Variant**, et est en lecture/écriture pour fermé [ensemble de cellules](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) objets et en lecture seule pour ouvrir **ensemble de cellules** objets. Le **Variant** doit contenir une valide **chaîne**, par exemple, une requête MDX.
