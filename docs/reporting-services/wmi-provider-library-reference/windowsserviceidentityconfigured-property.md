# Propri&#233;t&#233; WindowsServiceIdentityConfigured (WMI MSReportServer_ConfigurationSetting)
  Retourne l'identité sous laquelle le service Windows Report Server a été configuré la dernière fois pour son exécution. En lecture seule.  
  
## Syntaxe  
  
```vb  
Public Dim WindowsServiceIdentityConfigured As String  
```  
  
```csharp  
public string WindowsServiceIdentityConfigured;  
```  
  
## Valeurs de la propriété  
 Valeur **Chaîne** contenant l’identité sous laquelle le service Windows Report Server a été configuré la dernière fois pour son exécution.
