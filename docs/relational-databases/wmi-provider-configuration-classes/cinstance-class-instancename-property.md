# <a name="cinstance-class---instancename-property"></a>Classe CInstance - InstanceName, propriété
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Obtient le nom de l’instance de la [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] client.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object.InstanceName [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *objet*  
 Objet de [classe CInstance](../../relational-databases/wmi-provider-configuration-classes/cinstance-class.md) qui représente une instance du client [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Valeur de chaîne qui spécifie le nom de l'instance du client [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .
