# Propri&#233;t&#233;s de l&#39;abonnement - &lt;Abonnement&gt;
  Cette section fournit des informations sur la boîte de dialogue **Propriétés de l'abonnement** :  
  
-   [Propriétés de l’abonnement - abonné](../../relational-databases/replication/subscription-properties-subscriber.md) traite des propriétés d’abonnement disponibles sur l’abonné (abonnements par extraction de données uniquement).
