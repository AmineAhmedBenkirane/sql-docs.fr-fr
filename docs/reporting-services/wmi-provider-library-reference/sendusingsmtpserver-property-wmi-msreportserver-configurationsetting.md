# Propri&#233;t&#233; SendUsingSMTPServer (WMI MSReportServer_ConfigurationSetting)
  Spécifie si l’élément **SendUsing** dans la configuration de la messagerie pour le serveur de rapports est défini sur **true**.  
  
## Syntaxe  
  
```vb  
Public Dim SendUsingSMTPServer As Boolean  
```  
  
```csharp  
public Boolean SendUsingSMTPServer;  
```  
  
## Valeurs de la propriété  
 Objet **booléen** en lecture seule.
