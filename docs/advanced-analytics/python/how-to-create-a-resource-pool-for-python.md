# <a name="how-to-create-a-resource-pool-for-python"></a>Comment créer un pool de ressources pour Python
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md-winonly](../../includes/appliesto-ss-xxxx-xxxx-xxx-md-winonly.md)]

Cette section est en cours de développement.

> [!NOTE]
> Prise en charge de Python est une nouvelle fonctionnalité de SQL Server 2017 et dans la version préliminaire. Rechercher des informations plus rapidement.

En règle générale, vous pouvez analyser n’importe quel script externe, y compris un qui s’exécute de Python, à l’aide de la même infrastructure que celle qui a été fournie pour la gouvernance de ressources de scripts R dans SQL Server 2016. Pour plus d’informations, consultez [gouvernance des ressources pour R](../r/resource-governance-for-r-services.md).
