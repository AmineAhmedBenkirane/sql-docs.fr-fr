# Propri&#233;t&#233; VirtualDirectoryReportManager (WMI MSReportServer_ConfigurationSetting)
  Retourne le répertoire virtuel de l'application Gestionnaire de rapports. En lecture seule.  
  
## Syntaxe  
  
```vb  
Public Dim VirtualDirectoryReportManager As String  
```  
  
```csharp  
public string VirtualDirectoryReportManager;  
```  
  
## Valeurs de la propriété  
 **Chaîne** qui contient le répertoire virtuel.
