# <a name="sql-server-2012-sp3-release-notes"></a>Notes de publication de SQL Server 2012 SP3
Ces Notes de publication décrivent les problèmes que vous devez savoir avant d’installer ou de dépanner SQL Server 2012 Service Pack 3. Les notes de publication sont disponibles uniquement en ligne, et non sur un support d’installation. Elles sont mises à jour régulièrement à mesure que des problèmes sont identifiés. Pour obtenir la liste o améliorations et correctifs, consultez [informations de version de SQL Server 2012 Service Pack 3](http://support.microsoft.com/help/3072779/sql-server-2012-service-pack-3-release-information) pour plus d’informations.  
  
## <a name="choose-the-correct-file-to-download-and-install"></a>Choisir le fichier approprié à télécharger et installer  
Utilisez le tableau suivant pour identifier l'emplacement et le nom du fichier à télécharger en fonction de la version installée actuellement. Les pages de téléchargement indiquent la configuration système requise et contiennent des instructions d'installation de base.  

##<a name="download-pages"></a>Pages de téléchargement
Le lien suivant pour les principaux télécharger les packages pour SQL Server 2012 SP3.
- [SQL Server 2012 SP3](http://go.microsoft.com/fwlink/?linkid=615935)
- [SQL Server 2012 Express Service Pack 3](http://go.microsoft.com/fwlink/?linkid=692144)
- [Microsoft SQL Server 2012 SP3 Feature Pack](http://go.microsoft.com/fwlink/?linkid=615941)

Pour plus d’informations pour identifier l’emplacement et le nom du fichier à télécharger en fonction de la version actuellement installée, consultez la section « Sélectionner le fichier approprié à télécharger » dans [informations de version de SQL Server 2012 Service Pack 3](https://support.microsoft.com/en-us/help/3072779/sql-server-2012-service-pack-3-release-information).
