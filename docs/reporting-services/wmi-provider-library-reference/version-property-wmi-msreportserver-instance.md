# Propri&#233;t&#233; Version (WMI MSReportServer_Instance)
  Retourne la version du serveur de rapports au format Major.Minor.Build.Revision. En lecture seule.  
  
## Syntaxe  
  
```vb  
Public Dim Version As String  
```  
  
```csharp  
public string Version;  
```  
  
## Valeur de propriété  
 Valeur de type **string** qui contient la version du serveur de rapports.
