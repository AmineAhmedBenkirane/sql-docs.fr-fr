# <a name="flagvalue-property-serversettingsgeneralflag-class"></a>Propriété FlagValue (classe ServerSettingsGeneralFlag)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Obtient la valeur de l’indicateur.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object.FlagValue [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *objet*  
 Objet de [classe ServerSettingsGeneralFlag](../../../relational-databases/wmi-provider-configuration-classes/serversettingsgeneralflag-class/serversettingsgeneralflag-class.md) qui représente un indicateur général pour les paramètres du serveur.  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Valeur booléenne qui spécifie la valeur de l'indicateur.
