# <a name="customizing-calls-order-sybasetosql"></a>Personnalisation des appels, commande (SybaseToSQL)
Au cours de cette étape, vous sélectionnez l’ordre dans lequel les objets à tester seront appelés.  
  
## <a name="customizing-order"></a>Personnalisation de commande  
Utilisez les boutons placés en haut de la grille avec une liste d’objets à placer les objets dans le bon ordre. Vous pouvez également utiliser boutons apparaissent sur la droite lorsque la ligne est sélectionnée.
