# <a name="odbc-errors"></a>Erreurs ODBC
Lorsqu’une erreur se produit, les pilotes de base de données de bureau Microsoft ODBC retourne le numéro d’erreur native, la valeur SQLSTATE (un code d’erreur ODBC) et un message d’erreur.
