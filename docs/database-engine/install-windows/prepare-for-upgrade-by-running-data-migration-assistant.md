# Pr&#233;parer la mise &#224; niveau en ex&#233;cutant le Conseiller de mise &#224; niveau
  Préparez la mise à niveau de vos bases de données OLTP et l’adoption des nouvelles fonctionnalités de [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)] en téléchargeant et en exécutant Microsoft Data Migration Assistant.  
  
## <a name="download-and-install-data-migration-assistant"></a>Télécharger et installer Data Migration Assistant  
 Téléchargez et installez Data Migration Assistant à partir [d’ici](http://go.microsoft.com/fwlink/?LinkID=613421). Cet outil n’est pas fourni sur le média d’installation de [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)] .
