# <a name="annotations-master-data-services"></a>Annotations (Master Data Services)

[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md-winonly](../includes/appliesto-ss-xxxx-xxxx-xxx-md-winonly.md)]

  Dans [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)], les annotations sont des commentaires que vous entrez pour fournir des informations sur des transactions.  
  
 Vous pouvez annoter une transaction pour fournir des informations concernant la raison pour laquelle une action a été effectuée. Vous devez sélectionner une transaction pour ajouter une annotation associée. Les annotations sont affichées avec les transactions dans :  
  
-   La zone fonctionnelle **Explorateur** de l’application web [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)] .  
  
-   La zone fonctionnelle **Gestion des versions** de l’application web [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)] .  
  
-   [!INCLUDE[ssMDSXLS](../includes/ssmdsxls-md.md)]MDS.
