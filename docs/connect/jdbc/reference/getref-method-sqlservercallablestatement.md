# <a name="getref-method-sqlservercallablestatement"></a>Méthode getRef (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre désigné en tant qu’objet Ref dans le langage de programmation Java.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[getRef (int)](../../../connect/jdbc/reference/getref-method-int.md)|Récupère la valeur du paramètre désigné en tant qu’objet dans le langage de programmation Java fonction de l’index de paramètre Ref.|  
|[getRef (java.lang.String)](../../../connect/jdbc/reference/getref-method-java-lang-string.md)|Récupère la valeur du paramètre désigné en tant qu’objet dans le langage de programmation Java selon le nom de paramètre Ref.|
