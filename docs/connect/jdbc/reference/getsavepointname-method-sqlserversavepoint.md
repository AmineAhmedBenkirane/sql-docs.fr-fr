# <a name="getsavepointname-method-sqlserversavepoint"></a>Méthode getSavepointName (SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Obtient le nom du point de sauvegarde.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public java.lang.String getSavepointName()  
```  
  
## <a name="return-value"></a>Valeur retournée  
 A **chaîne** qui contient le nom du point de sauvegarde.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode getSavepointName est spécifiée par la méthode getSavepointName dans l’interface java.sql.Savepoint.
