# Propri&#233;t&#233;s du serveur (page G&#233;n&#233;ral) - SQL Server Management Studio
  Utilisez cette page pour consulter des informations en lecture seule sur votre installation [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
## Grille des propriétés  
 Consultez les propriétés du serveur sélectionné, comme le nom du serveur, le système d'exploitation du serveur ou encore le nombre de processeurs.  
  
 **Nom**  
 Affiche le nom de l'instance du serveur.  
  
 **Product**  
 Affiche l'édition de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] en cours d'exécution.  
  
 **Système d'exploitation**  
 Affiche le système d'exploitation [!INCLUDE[msCoName](../../includes/msconame-md.md)] Windows en cours d'exécution.  
  
 **Plateforme**  
 Décrit le système d'exploitation et le matériel exécutant [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
 **Version**  
 Affiche le numéro de version de l'édition [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] en cours d'exécution.  
  
 **Langage**  
 Affiche la langue prise en charge par l'instance [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] en cours d'exécution.  
  
 **Mémoire**  
 Affiche la quantité de mémoire vive (RAM) installée sur le serveur.  
  
 **Processeurs**  
 Affiche le nombre d'unités centrales installées.  
  
 **Répertoire racine**  
 Affiche le chemin de l’emplacement de l’instance [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)], en général C:\Program Files\Microsoft SQL Server\\.  
  
 **Classement du serveur**  
 Affiche le classement pris en charge par le serveur. Un classement spécifie la page de codes et l'ordre de tri à utiliser pour les données Unicode et non-Unicode.  
  
 **Is Clustered**  
 Affiche **True** si l’instance de serveur est configurée dans un cluster de basculement [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] ou **False** si l’instance de serveur n’est pas en cluster.  
  
 **Is HADR Enabled**  
 Affiche **True** si la fonctionnalité [!INCLUDE[ssHADR](../../includes/sshadr-md.md)] est activée ou **False** si la fonctionnalité [!INCLUDE[ssHADR](../../includes/sshadr-md.md)] est désactivée. Pour plus d’informations sur l’activation ou la désactivation de [!INCLUDE[ssHADR](../../includes/sshadr-md.md)], consultez [Activer et désactiver les groupes de disponibilité Always On &#40;SQL Server&#41;](../../database-engine/availability-groups/windows/enable-and-disable-always-on-availability-groups-sql-server.md).
