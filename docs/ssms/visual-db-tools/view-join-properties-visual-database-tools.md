# <a name="view-join-properties-visual-database-tools"></a>Afficher les propriétés d'une jointure (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Vous pouvez consulter une collection de propriétés pour une vue dans la fenêtre Propriétés.  
  
### <a name="to-view-join-properties"></a>Pour afficher les propriétés d'une jointure  
  
1.  Cliquez avec le bouton droit sur la jointure dans le volet **Schéma** .  
  
2.  Dans le menu contextuel, cliquez sur **Propriétés**.  
  
    Les propriétés de la jointure s'affichent dans la fenêtre Propriétés.
