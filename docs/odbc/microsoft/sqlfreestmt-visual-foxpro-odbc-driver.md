# <a name="sqlfreestmt-visual-foxpro-odbc-driver"></a>SQLFreeStmt (le pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous [référence de l’API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : complet  
  
 Conformité d’API ODBC : Niveau principal  
  
 Arrête le traitement associé à un spécifique *hstmt*, ferme tous les curseurs ouverts, associés à la *hstmt*, ignore les résultats en attente et éventuellement libère toutes les ressources associées au handle d’instruction.  
  
 Pour plus d’informations, consultez [SQLFreeStmt](../../odbc/reference/syntax/sqlfreestmt-function.md) dans les *de référence du programmeur ODBC*.
