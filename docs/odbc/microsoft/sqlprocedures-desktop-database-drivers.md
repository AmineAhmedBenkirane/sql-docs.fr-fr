# <a name="sqlprocedures-desktop-database-drivers"></a>SQLProcedures (pilotes bureau de base de données)
**SQLProcedures** renvoie uniquement les lignes pour les procédures qui ont au moins un argument. Les procédures qui ont pas d’arguments sont traités en tant que vues.  
  
|Colonne|Commentaires|  
|------------|--------------|  
|PROCEDURE_QUALIFIER|Chemin d'accès au fichier de base de données.|  
|PROCEDURE_OWNER|NULL|  
|PROCEDURE_NAME|Nom de la procédure et sans délimitation|  
|PROCEDURE_TYPE|SQL_PT_PROCEDURE|
