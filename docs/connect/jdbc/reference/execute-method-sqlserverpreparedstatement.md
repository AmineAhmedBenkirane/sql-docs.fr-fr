# <a name="execute-method-sqlserverpreparedstatement"></a>Execute (méthode) (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Exécute l’instruction SQL dans cette [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) objet, ce qui peut être n’importe quel type d’instruction SQL.

|Nom| Description|  
|----------|-----------------|  
|[Exécutez ()](../../../connect/jdbc/reference/execute-method.md)|Exécute l’instruction SQL dans cette [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) objet, ce qui peut être n’importe quel type d’instruction SQL.|  
|[Execute (java.lang.String)](../../../connect/jdbc/reference/execute-method-java-lang-string.md)|Exécute l'instruction SQL donnée, qui peut retourner plusieurs résultats.|
