# Propri&#233;t&#233; EditionName (WMI MSReportServer_Instance)
  Retourne le nom explicite de l'édition du serveur de rapports. En lecture seule.  
  
## Syntaxe  
  
```vb  
Public Dim EditionName As String  
```  
  
```csharp  
public string EditionName;  
```  
  
## Valeur de propriété  
 Chaîne (**string**) qui contient le nom lisible de l’édition du serveur de rapports.
