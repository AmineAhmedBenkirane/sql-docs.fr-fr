# R Server (Standalone)
  **Microsoft R serveur (autonome)** fait partie de la plateforme d’analytique entreprise fournie par les Services Microsoft R.  
  
 Il étend R en fournissant la prise en charge, l’évolutivité et la sécurité. [!INCLUDE[rsql_platform](../../includes/rsql-platform-md.md)] résout également les limitations de mémoire d’open source de R en ajoutant parallèle et de traitement mémorisé en bloc des données, permettant aux utilisateurs d’analyser les données à l’échelle de l’entreprise et utiliser des données beaucoup plus volumineuses que peut tenir dans la mémoire.  
  
  
## Prise en main de Microsoft R Server  
  
 Pour plus d’informations sur la façon de configurer un serveur Windows pour prendre en charge les applications R, consultez [créer un Standalone R Server](../../advanced-analytics/r-services/create-a-standalone-r-server.md).  

Vous pouvez également configurer une machine virtuelle Azure qui comprend un serveur R (autonome) ainsi que DeployR, pour l’incorporation analytique R dans les applications et l’intégration de R avec les systèmes back-end. Pour plus d'informations, consultez :

+ [Approvisionner une Machine virtuelle du serveur R](../../advanced-analytics/r-services/provision-the-r-server-only-sql-server-2016-enterprise-vm-on-azure.md)
+ [Déployer R](https://msdn.microsoft.com/microsoft-r/deployr-about)
  
Pour plus d’informations sur Microsoft R, didacticiels et exemples, consultez [les produits Microsoft R](https://msdn.microsoft.com/microsoft-r/microsoft-r-getting-started).
